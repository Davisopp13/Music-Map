# Letterpress fonts

Source TTFs for the self-hosted MapLibre glyph PBFs in `public/glyphs/`
(regenerate with `npm run glyphs`).

| File            | Face                 | Used for                        |
| --------------- | -------------------- | ------------------------------- |
| oswald-600.ttf  | Oswald SemiBold      | city/town labels (wood type)    |
| oswald-500.ttf  | Oswald Medium        | neighborhoods, states           |
| lora-500.ttf    | Lora Medium          | street labels                   |
| lora-500i.ttf   | Lora Medium Italic   | water labels (atlas italic)     |

Both families are licensed under the SIL Open Font License 1.1 and were
downloaded from Google Fonts (fonts.gstatic.com static instances).
